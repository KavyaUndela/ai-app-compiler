# AI Compiler Dashboard

This is a minimal Next.js dashboard for the AI Application Compiler prototype. It includes pages for:

- Prompt Input
- Pipeline Viewer
- Validation Viewer
- Repair Viewer
- Runtime Preview
- Metrics Dashboard

Usage:

1. Install dependencies from `apps/web`:

```bash
cd "apps/web"
npm install
npm run dev
```

2. Ensure the backend FastAPI service is running at `http://localhost:8000` or set `NEXT_PUBLIC_API_URL`.
